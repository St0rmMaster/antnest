## Инструкция по заполнению

1. Скопируйте этот шаблон в основной README.md в раздел "Каталог сервисов/ботов"
2. Замените [имя-сервиса] на понятное название (формат: `stack-role`)
3. Заполните все поля в квадратных скобках актуальными данными
4. В разделе "Операции" опишите реальные шаги для вашего сервиса
5. Каждая операция должна содержать 3-7 шагов максимум
6. Удалите неприменимые поля или отметьте их как "N/A"
7. Обновите дату в поле "Обновлено"
8. Удалите эту инструкцию из итогового документа

## Примеры именования сервисов

- `nginx-proxy` — обратный прокси
- `postgres-main` — основная база данных
- `redis-cache` — кэш-сервер
- `telegram-bot` — телеграм бот
- `api-gateway` — шлюз API
- `monitoring-grafana` — система визуализации метрик
