# План проекта Antnest

## Цели проекта

Проект Antnest направлен на создание системы координации и оркестрации множества специализированных ИИ-агентов.

## Текущее состояние

### Реализовано

- **Nestor Orchestrator** - базовый сервис-оркестратор
  - FastAPI API с эндпоинтами для работы с задачами
  - Граф LangGraph для обработки задач
  - Интеграция с OpenAI LLM для генерации планов и вопросов
  - Базовая структура состояния задачи

## Планы развития

### Краткосрочные (ближайшие итерации)

- Расширение графа обработки задач
- Добавление узлов для выполнения конкретных действий
- Интеграция с дополнительными LLM-провайдерами
- Улучшение обработки ошибок и логирования

### Среднесрочные

- Добавление персистентности состояния задач
- Реализация механизма координации между агентами
- Создание системы приоритизации задач
- Добавление метрик и мониторинга

### Долгосрочные

- Полноценная система оркестрации множества агентов
- Поддержка различных типов задач и агентов
- Масштабирование и оптимизация производительности
- Расширенная аналитика и отчётность

## Архитектура

### Nestor Orchestrator

Центральный узел системы, отвечающий за:
- Приём и обработку задач
- Генерацию планов выполнения
- Координацию работы агентов
- Управление состоянием задач

## Технологический стек

- Python 3.10+
- FastAPI - веб-фреймворк
- LangGraph - построение графов состояний
- LangChain OpenAI - интеграция с LLM
- Uvicorn - ASGI-сервер
